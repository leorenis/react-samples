## Webpack Training

Training webpack 5. 

### Setup
- Webpack
- Webpack CLI
- EcmaScript

Using node 22 by `.nvmrc`
```bash
nvm use
```

```bash
npm install --save-dev webpack@5.97.1
```

```bash
npm install --save-dev webpack-cli@6.0.1
```

### Running webpack

To running webpack `--mode development`

```zsh
npx webpack --mode development
```

Production mode
```zsh
npx webpack --mode development
```

```zsh
npx webpack --mode production
```
